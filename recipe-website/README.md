# 🍛 Indian Recipe Website

A simple and responsive **multi-page recipe website** built using **HTML and CSS**.

This project contains multiple Indian vegetarian recipes with clean layout, navigation links, responsive design, hover effects, and improved typography.

---

## 📌 Features

- 🏠 Multi-page website
- 🍛 Multiple Indian recipes
- 📱 Responsive design using media queries
- 🎨 Styled with CSS
- ✨ Hover animations
- 🖼️ Responsive images
- 🧭 Navigation bar between pages
- 📋 Table of contents links
- 🦶 Footer section
- 🌱 Pure vegetarian recipe website

---

## 📂 Pages Included

- `index.html` → Home page
- `recipe1.html` → Vegetable Dum Biryani
- `recipe2.html` → Naan Recipe
- `recipe3.html` → Kala Chana
- `recipe4.html` → Chickpea Curry
- `recipe5.html` → Masala Raita
